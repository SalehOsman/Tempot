## Introduction

Welcome to the Tempot bot user guide. This guide explains how to get started with the bot and the basic commands available to you.

Arabic is the primary language of the bot, with full English language support.

## Getting Started with the Bot

### Step 1: Find the Bot

Search for the bot on Telegram using the username provided by your system operator, or use the direct link you received.

### Step 2: Start a Conversation

Press the **"Start"** button or send the `/start` command to begin interacting with the bot.

### Step 3: Choose Your Language

The bot will ask you to select your interface language:

- **العربية** — Arabic (primary)
- **English**

You can change the language at any time later.

## Basic Commands

| Command     | Description                    |
| ----------- | ------------------------------ |
| `/start`    | Start interacting with the bot |
| `/help`     | Display available commands     |
| `/settings` | Open user settings             |
| `/language` | Change interface language      |

## Navigating Menus

The bot uses interactive buttons (Inline Keyboard) for navigation. Press buttons to select options instead of typing commands manually.

## Sessions and Conversations

The bot remembers your conversation context. If you are in the middle of a process (such as filling out a form), you can:

- Complete the process by sending the required data
- Cancel the current process using the "Cancel" button
- Return to the main menu using `/start`

## Roles and Permissions

The bot supports four permission levels:

| Role            | Description                       |
| --------------- | --------------------------------- |
| **Guest**       | Limited access to public features |
| **User**        | Full access to core features      |
| **Admin**       | Content and user management       |
| **Super Admin** | Full control over all settings    |

Your role is assigned by the system operator. Contact them if you need additional permissions.

## Help and Support

If you encounter an issue:

1. Use the `/help` command to get information about available commands
2. Verify the bot is working by sending `/start`
3. Contact your system operator for technical assistance
